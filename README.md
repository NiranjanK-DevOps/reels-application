# Reels Application

A simple reels-style video scrolling interface built using HTML, CSS, and JavaScript. Videos are hosted on AWS S3.

## Features
- Auto-play videos when visible
- Pause videos when out of view
- Smooth vertical scrolling UX
- S3-hosted video streaming
- Responsive layout

## Technologies Used
- HTML / CSS / JavaScript
- AWS S3 (Video hosting)

## Purpose of the Project
This project was built to understand:
- How S3 can serve video files
- Scroll-based visibility logic
- Lightweight frontend interaction
